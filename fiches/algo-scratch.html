Interface Scratch
🎬 Zone de scène

C'est ici que le programme s'exécute et que les lutins bougent. Elle mesure 480 × 360 pixels. Le centre est le point (0, 0).

🧩 Zone des blocs

Palette de tous les blocs disponibles, classés par catégories colorées. On les fait glisser dans la zone de scripts.

📜 Zone de scripts

On assemble les blocs ici pour créer le programme du lutin sélectionné. Chaque lutin a ses propres scripts.

Les 8 catégories de blocs
🔵 Mouvement
Avancer, tourner, aller à (x,y), rebondir
🟣 Apparence
Dire, changer costume, afficher/cacher
🔊 Son
Jouer un son, régler le volume
⚙️ Événements
Quand ▶ cliqué, Quand touche pressée
🔄 Contrôle
Répéter, Si…alors, Attendre, Arrêter
🔍 Capteurs
Touche la couleur ?, Demander, Timer
🔢 Opérateurs
+, −, ×, ÷, mod, =, <, >, et, ou
📦 Variables
Créer une variable, Mettre à, Ajouter à
Blocs essentiels à connaître

ÉVÉNEMENTS (chapeau orange)

Quand ▶ est cliqué

Démarre le script quand on clique sur le drapeau vert

Quand la touche [espace] est pressée

Déclenche le script avec une touche du clavier

CONTRÔLE (orange foncé)

Répéter 10 fois … fin

Boucle bornée — répète N fois

Répéter indéfiniment … fin

Boucle infinie — s'arrête avec "Arrêter tout"

Si <condition> alors … fin

Alternative — exécute le bloc si la condition est vraie

Lutin vs Scène : chaque lutin a ses propres scripts. La scène peut aussi avoir des scripts (fond musical, score affiché…)
Coordonnées (x, y)

La scène va de −240 à +240 en x et de −180 à +180 en y. Le centre (0, 0) est au milieu.

Variables

Créer dans "Variables" → "Créer une variable". Utiliser Mettre à pour affecter, Ajouter à pour incrémenter.

Diffuser un message

Permet de faire communiquer les lutins entre eux. Bloc Envoyer à tous + Quand je reçois.
