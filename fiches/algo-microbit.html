La carte micro:bit — composants principaux
💡 Afficheur 5×5

25 LEDs disposées en grille. Peut afficher des icônes, du texte défilant, des nombres ou des animations.

🔘 Boutons A et B

Deux boutons poussoirs. Détectables individuellement ou en combinaison (A+B ensemble).

📡 Capteurs intégrés

Accéléromètre (mouvement, secousse), boussole, thermomètre, lumière, son (v2).

🔌 Broches (pins)

P0, P1, P2 + alimentation (3V, GND). Permettent de brancher des capteurs et actionneurs externes.

Environnement MakeCode

Site : makecode.microbit.org — pas d'installation requise.

Blocs → JavaScript → Python

MakeCode propose 3 modes : blocs graphiques, JavaScript et Python. On peut passer de l'un à l'autre à tout moment.

Téléverser le programme
  1. Cliquer sur Télécharger → fichier .hex
  2. Brancher le micro:bit en USB
  3. Glisser le fichier dans le lecteur MICROBIT
  4. La LED orange clignote, puis le programme démarre

Utiliser les broches — capteurs et actionneurs externes
Lire une broche (capteur)
# Python MicroPython
from microbit import *

while True:
    val = pin0.read_analog()  # 0–1023
    if val > 500:
        display.show(Image.YES)
    else:
        display.show(Image.NO)
    sleep(200)
Écrire sur une broche (actionneur)
# Allumer une LED externe
pin1.write_digital(1)  # allumé
sleep(1000)
pin1.write_digital(0)  # éteint

# Signal analogique (PWM)
pin0.write_analog(512) # 0–1023
Lecture numérique
pin0.read_digital()
Retourne 0 ou 1
Lecture analogique
pin0.read_analog()
Retourne 0 à 1023
Écriture numérique
pin1.write_digital(1)
0 = éteint, 1 = allumé
Écriture analogique
pin1.write_analog(512)
0 à 1023 (PWM)
Fonctions utiles
  • temperature() — °C interne
  • accelerometer.get_x()
  • compass.heading() — boussole
  • sleep(ms) — pause en ms
Affichage
  • display.show("Hello")
  • display.show(42)
  • display.show(Image.HEART)
  • display.clear()
  • display.set_pixel(x,y,b)
Branchement GND

Tout capteur ou actionneur doit être relié à GND (masse). L'alimentation se prend sur 3V (max 90 mA au total).
